# 육아정보 서비스 운영 정책

최종 업데이트: 2025-05

---

## 1. 콘텐츠 생성 정책

### 1.1 품질 기준

- 모든 콘텐츠는 GPT-4o-mini로 초안 생성 후 자체 품질 검수 과정을 거친다
- 100점 만점 기준 **85점 이상**만 DB에 게시
- 85점 미만 시 힌트를 보강해 최대 **2회 재생성** 후 그래도 미달이면 해당 회차 건너뜀

### 1.2 평가 기준 (각 20점)

| 항목 | 설명 |
|------|------|
| 정확성 | 의학적·과학적으로 신뢰할 수 있는가 |
| 완성도 | 내용이 충분하고 구체적인가 |
| 실용성 | 실제 육아에 바로 적용 가능한가 |
| 안전성 | 잘못된 정보로 위험을 줄 수 있는가 (역점수) |
| 가독성 | 이해하기 쉽고 구조화되어 있는가 |

### 1.3 의료 정보 원칙

- 콘텐츠는 **참고용**이며 의료 전문가의 조언을 대체하지 않는다
- 진단·처방·투약에 관한 구체적 지침은 제공하지 않는다
- 서비스 내 모든 페이지 하단에 의료 고지 문구를 표시한다:  
  > "이 서비스의 정보는 참고용이며 의료 전문가의 조언을 대체하지 않아요"
- 의심스러운 내용은 어드민에서 수동 삭제 후 재생성한다

### 1.4 중복 방지

- 신규 생성 시 기존 글의 제목·요약·태그를 GPT 컨텍스트로 전달해 중복 주제 생성 억제
- 같은 카테고리×토픽 조합에서 글이 적은 순으로 우선 생성 (커버리지 균등화)

## 2. 브랜드·제품 언급 정책

### 2.1 허용 기준

다음 조건을 **모두** 충족하는 브랜드만 콘텐츠에 언급한다:

1. 한국에서 실제 구매 가능 (쿠팡·네이버쇼핑·롯데마트·이마트 등 국내 채널)
2. 국내 소비자 인지도가 있는 브랜드
3. 영유아 안전 기준(KC 인증 등)을 충족하는 제품

### 2.2 금지 사항

- 국내 미유통·미판매 해외 브랜드 단독 언급
- 특정 쇼핑몰·판매처에 대한 직접 링크 또는 광고성 추천
- 의료기기에 준하는 제품의 효능·효과 주장

### 2.3 브랜드 관련 기존 글 수정

- 브랜드 정책 위반 글은 **삭제하지 않고 UPDATE**로 수정 (ID·slug·투표 데이터 유지)
- `scripts/update-brand-articles.ts` 패턴 사용

## 3. 사용자 데이터 정책

### 3.1 수집 데이터 없음

- 로그인·회원가입 없음
- 개인정보 수집 없음
- 외부 분석 도구(Google Analytics 등) 미사용

### 3.2 로컬 저장 전용

사용자 기기의 `localStorage`에만 저장되는 항목:

| 키 | 내용 |
|----|------|
| `pregnancy-calculator-v1` | 임신 주차 계산기 입력값 |
| `vote-v1-{articleId}` | 도움됐나요 투표 여부 (중복 방지) |
| `checklist-*` | 출산 준비물 체크 상태 |

- 서버에 전송되지 않음
- 브라우저 데이터 삭제 시 함께 삭제됨

### 3.3 투표 데이터

- "도움됐나요" 투표 결과(helpful_count, not_helpful_count)는 Supabase에 집계 형태로만 저장
- 어떤 사용자가 투표했는지 식별 불가

## 4. 광고 정책

- **광고 없음** — 서비스 내 어떠한 형태의 광고도 게재하지 않는다
- 백색소음기는 Web Audio API로 소리를 직접 생성해 외부 오디오 파일·서비스 의존 없음
- 콘텐츠 내 브랜드 언급은 편의 정보 제공 목적이며 광고비를 받지 않는다

## 5. 자동화 운영 정책

### 5.1 일일 파이프라인

- 매일 KST 06:00 GitHub Actions로 자동 실행
- 하루 최대 10개 글 생성
- 파이프라인 로그는 어드민 대시보드에서 확인 가능

### 5.2 운영자 역할

- 하루 1회 어드민 대시보드 확인
- 품질 미달 또는 부적절한 콘텐츠 수동 삭제
- 필요 시 수동 콘텐츠 생성 (어드민 폼 사용)

### 5.3 장애 대응

- GitHub Actions 실패 시 다음날 재실행 (별도 알림 없음)
- Supabase·Vercel 장애 시 서비스 자연 중단 — 별도 페일오버 없음
- 운영 비용 목표: 월 $0~$10 이하

## 6. 콘텐츠 삭제 정책

- 의료적으로 위험한 정보, 사실 오류, 브랜드 정책 위반 시 어드민에서 즉시 삭제
- 삭제 시 해당 pipeline_log의 article_id는 NULL로 자동 설정됨 (ON DELETE SET NULL)
- 삭제 후 동일 카테고리×토픽 조합은 다음 파이프라인 사이클에 자동 재생성
